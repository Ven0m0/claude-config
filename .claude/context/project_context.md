# Project Context

This directory contains project-specific context that can be provided to Claude to improve responses.

## Usage

Add files here that contain:
- Project background and goals
- Technical specifications
- Team conventions and standards
- Domain-specific knowledge
- Common patterns and practices

## Example Files

- `architecture.md`: System architecture documentation
- `conventions.md`: Coding conventions and style guides
- `glossary.md`: Project-specific terminology
- `dependencies.md`: Project dependencies and their usage
