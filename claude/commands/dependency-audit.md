---
description: Audit dependencies for security vulnerabilities
category: security-audit
---

Perform dependency security audit:

1. Identify dependency files (package.json, requirements.txt, Cargo.toml, etc.)
1. Run security audit: `npm audit`, `uvx pip-audit`, `cargo audit`
1. Check for outdated packages and available updates
1. Analyze severity levels and CVE references
1. Create prioritized update plan with remediation steps
1. Document findings and recommendations
